# Contributing

Thank you for your interest in this project.

This repository serves as a **read-only companion** to a published research paper. Its purpose is to ensure transparency and reproducibility of the results.

## Policy

- **Pull requests** are not accepted. The code and data are provided as-is to accompany the paper.
- **Issues** are welcome for reporting bugs, asking questions about the methodology, or requesting clarifications.

## Reuse

You are free to **fork** this repository and adapt the code for your own research, provided you comply with the [CC BY-NC 4.0 license](LICENSE) (attribution required, non-commercial use only).

If you build upon this work, please cite the original paper (see [CITATION.cff](CITATION.cff)).

## Contact

For questions or collaboration inquiries, please open an issue or contact the corresponding author at rodrigo.demarco@usm.cl.
